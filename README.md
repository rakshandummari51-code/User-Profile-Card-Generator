# User Profile Card Generator

## Overview
This project is a form-based web application built using Python Flask. Users can enter profile details such as name, bio, and image URL to generate a formatted user profile card dynamically.

## Features
- User input form
- Dynamic profile card generation
- Backend form processing
- Frontend data display

## Technologies Used
- Python
- Flask
- HTML
